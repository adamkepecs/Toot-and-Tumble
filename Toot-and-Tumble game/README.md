# Toot and Tumble

Construct 3 MVP folder project.

Open this folder in Construct 3 with **Menu -> Project -> Open -> Project folder** and choose this directory, or zip the folder contents and rename the zip to `.c3p`.

Controls:
- Space / mouse / touch press: freeze the spinning trombone angle
- Release: blast opposite the horn direction
- R / tap fail or win screen: restart

There is no normal jump. The trombone is the jump.

Visual cue: bright green tops are safe landing surfaces, cyan dashed platforms are jump-through, and red striped/glowing objects are hazards.
